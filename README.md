DESCRIPTION OF THE COURSE

  Intro to Data Science and Programming is a course that has taught us the beginner concepts in data science overarching 
  general themes of the field. It is a course designed to give us a strong base of knowledge to continue a journey working
  in the data science field. Through several weeks of different programming lessons, we have been able to thoroughly explore the 
  core tasks of description, prediction, and causal inference. Through this foundational exploration we look at data science and 
  programming through a lens targeted toward the healthcare industry and the many valuable applications this field can have in
  healthcare and research studies. We are also able to take a look into concepts like machine learning, best practices in the field,
  and the many different ways that our experience in this course can help us in our careers and the real-world in general. 

WHAT I LEARNED

  The main points of this course were the three core tasks of data science: Description, Prediction, and Causal Inference. Among these 
  tasks there were many other parts of data science that we covered, which allowed us to see the bigger picture in a clearer light. To begin with, 
  we learned basic concepts of data storage and retrieval in SQL. As our first language it was good because it was fairly intuitive. 
  We then got into the description task. We began this task in SQL by cleaning data and summarizing it. We then switched over to Python and did the 
  same type of tasks. We then got into Data Visualization, creating many different ways to see data in a more approachable way, such as bar graphs,
  line graphs, and even violin graphs. We were able to do this because we imported libraries like pandas to aid in our coding. We then moved 
  through dashboards which allowed us to use many different types of data visualization at once. This was especially interesting as it seemed to 
  have a great deal of practical use in many fields in which we could be working. We then moved onto our prediction task. This becan with an
  introduction to machine learning and how we can prepare our data to be used in a machine learning model. We then began building models in Python 
  with the help of several different libraries.
